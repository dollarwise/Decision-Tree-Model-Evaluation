


# Decision Tree Classification – Airline Customer Satisfaction

## Project Overview

This project builds a Decision Tree classifier to predict airline customer satisfaction using passenger survey data.

## Dataset

Invistico Airline Customer Satisfaction Dataset

Rows: 129,880

Target Variable:
- satisfaction

## Methodology

1. Data preprocessing
2. Train-test split
3. Hyperparameter tuning with GridSearchCV
4. Decision Tree model training
5. Model evaluation using:
   - Confusion Matrix
   - Classification Report
   - F1-score
6. Decision Tree visualization
7. Feature importance analysis

## Best Hyperparameters

Obtained using GridSearchCV.

## Top Drivers of Satisfaction

- Seat Comfort
- In-flight Wifi Service
- Inflight Entertainment
- Online Boarding
- Leg Room Service

## Business Insights

Improving seating comfort, entertainment systems, Wi-Fi quality, and boarding experience can significantly improve customer satisfaction.

## Decision Tree vs Logistic Regression

Decision Trees are highly interpretable and capture nonlinear relationships, making them suitable for operational decision-making.
